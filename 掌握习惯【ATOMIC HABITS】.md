####  一、行为转变的三个层次：

1. Outcomes结果：行为转变所带来的附属品，不能以结果为目标而去改变习惯

2. Progresses进展：改变行为的具体实施方案

3. Identity身份：以想达到的身份为目标会更有动力去改变行为，为了达到这个身份可以促进自己在点点滴滴中改变行为

#### 二、好习惯行程的四个定律：

1. 提示（让它显而易见）
   1. 填写“习惯记分卡”；记下你当前的习惯并留意它们，分清楚好习惯、中性习惯、坏习惯
   2. 应用执行意图：我将于【时间】在【地点】【行为】，类似于smart原则，更有利于好习惯的执行
   3. 应用习惯叠加：把新习惯于老习惯关联起来，更有利于新习惯的养成
   4. 设计环境，让好习惯提示更清晰，新环境也更利用新习惯的养成

2. 渴望（让它有吸引力）
     1）利用喜好绑定。用你喜欢的行为强化你需要的动作，类似《自控力》的多巴胺利用
     2）加入把你喜欢的行为视为正常行为的文化群里，类似《自控力》群体对个体意志力的影响
     3）创设一种激励仪式，在实施低频行动之前先做一件你特别喜欢的事

3. 反应（让它简便易行）
   1. 减小阻力。减少培养好习惯的步骤
   2. 备好环境。创造一种有利于未来行为的环境
   3. 把握好决定性时刻。优化可以产生重大影响的小选择
   4. 利用两分钟准则。缩短你的习惯所占用的时间，争取只需要两分钟，甚至更少
   5. 自动化你的习惯。在能够锁定你未来行为的技术和物品上有所投入

4. 奖励（让它令人愉悦）
   1. 利用增强法。完成一套习惯动作后立即奖励自己
   2. 让“无所事事”变得愉悦。当避免坏习惯时，设计一种让由此带来的好处显而易见的方式
   3. 利用习惯追踪。记录习惯倾向，不要中断
   4. 决不连续错过两次。如果你忘了做，一定要尽快补救

#### 三、戒除坏习惯的四个定律：

1. 让它脱离视线
     降低出现频率。把坏习惯的提示清除出你所在的坏境

2. 让它缺乏吸引力
     重新梳理你的思路。罗列出戒除坏习惯所带来的的益处

3. 让它难以施行
   1. 增大阻力。增加实行坏习惯的步骤
   2. 利用承诺机制。锁定未来会有利于你的选择

4. 让它令人厌恶
   1. 找一个问责伙伴。情人监督你的行为
   2. 创立习惯契约。让坏习惯的恶果公开化，并令人难以忍受

#### 四、高级战术

1. 了解自己，做擅长的事，更容易达到心流，也更容易把擅长的事培养成习惯
2. 金发女孩准则：任务难度比自己高出4%，更容易激发心流，任务难度太低太无聊，任务难度太高容易放弃
3. .当习惯变成常规后容易产生倦怠，只有按照计划行事毫不动摇，才能保持专注
4. 当习惯固定后不容易改变，但是身份目标是灵活的，随着每个阶段身份目标的不同，要对习惯进行微调
